CheatSheet

Openen en aanmaken van een map in de command Prompt (cmd):

1. Ga naar Windows.

2. Ga naar de zoekbalk.

3. Typ cmd in en druk op enter.

  • 3a. De command Prompt opent.
  • 3b. Met het commando cd kan je kan je naar de volgende map gaan.
  • 3c. Met het commando cd .. ga je een mapje terug.
  • 3d. Als je wil weten waar je mappen staan dan kan je het commando dir gebruiken.
  • 3e. Om een nieuwe map aan te maken kan je het commando mkdir. (bestand aanmaken) gebruiken.

Een snellere manier:

1. Klik het icoon bestanden aan.

2. Zoek het mapje op waar een extra map moet worden toegevoegd.

3. Klik in de adresbar en typ daar cmd/enter.

  • 3a. Hiermee kun je de cd stap voortaan overslaan.


Een tekstbestand aanmaken:

1. Ga naar je map.

  • 1a. Vind je dat het cd commando te lang duurt, doe het dan op de snellere manier).

2. Klik in de adresbar en typ daar cmd/enter.

3. Vul notepad test.txt en druk op enter (test is optioneel, je kunt hier een eigen naam invullen).

4. Je ziet een PopUp van Notepad met de tekst: “Cannot find the test.txt file. Do you want create a new file? Yes or No.”

5. Klik op Yes.

6. Creeër je tekst.

7. Sla je het document op met Ctrl S.


Het aanmaken van een repository:

1. Ga in cmd naar de map waarin het notepad bestand is opgeslagen.

2. Typ git status.

  • 2a. Je bestand wordt zichtbaar in rood.

3. Typ het commando git init in.

3a. Dit commando zorgt ervoor dat een git bestand wordt aangemaakt.

4. Typ het commando git status in en druk op enter.

  • 4a. Je ziet main of master.

  • 4b. Als master verschijnt: Typ git branch -m main/enter. (Resultaat: Main verschijnt)

  • 4c. Als main verschijnt: Je kunt door naar de volgende stap.

  • 4d. Er kan een PopUp verschijnen waarin word gevraagd naar mail en Username.

  • 4e. In dat geval: kopieer deze vraag en plak deze vraag in de onderste regel.

  • 4f. Vervang de mail & username in de onderste regel met je eigen mail & naam.


Bestand zetten in de repository:

1. Mocht je nog geen Notepad hebben aangemaakt, maak er dan een aan (zie “Een tekstbestand aanmaken” ).

2. Typ git status.

  • 2a. De tekst in de cmd kan rood zijn. (Het bestand die je heb gemaakt in notepad is nieuw of aangepast.)

3. Typ (als de test rood is) typ git add . Klik op enter.

  • 3a. Je zorgt er dan voor dat hij opgeslagen wordt in de cmd.

4. Controleer of de tekst groen is geworden met git status.

5. Typ git commit -m “naam”.

  • 5a. Dat zorgt ervoor dat het in de cmd word opgeslagen.

6. Typ daarna weer git status in.


Updaten van de repository:

1. Je moet nu kunnen zien dat de tekst in Notepad in main zit.

  • 1c. Heb je iets veranderd in Notepad volg: (‘Bestand zetten in de repository’).

  • 1d. Typ git status/enter.

  • 1e. Typ git commit -m “naam” en klik op enter.

  • 1f. Typ git status en klik op enter.

2. Met git log & git log --oneline kan je aanpassingen zien in een rij.

  • 2a. Je hoeft het dus ook niet te gebruiken. Het mag.


GitHub account aanmaken & Upload Notepad het in GitHub.

1. Maakt een nieuw account aan in GitHub.

2. Maak een nieuw Repository aan in GitHub.

3. Ga naar de pagina in GitHub die je heb geopent.

5. Je ziet (‘...or push an exiting repository from the command line’).

  • 5a. Kopieer de eerste zin met de mail.

  • 5b. Plak het in je cmd.

  • 5c. Staat het al in main? (tweede link hoeft niet)

6. Alles groen?

  • 6a. Typ git push -u origin main/enter.

7. Refesh Github.

8. Dan staat je Notepad gegevens in Github.
